GatomIA is a VS Code extension that supercharges your development workflow with Agentic Spec-Driven Development. It connects GitHub Copilot Chat with structured specifications, steering documents, custom agents, and automation hooks — so you can plan features, generate code, and automate repetitive tasks without leaving your editor.

## What GatomIA does

GatomIA organizes your AI-assisted development around four core pillars:

<Steps>
  <Step title="Specify">
    Define features as structured specs using SpecKit or OpenSpec. The **Specs** view lets you create, browse, and manage specifications from a dedicated sidebar panel.
  </Step>
  <Step title="Implement">
    Open a generated `tasks.md` file and click **Start Task** to send implementation context directly to GitHub Copilot Chat. Tasks are tracked and can be dispatched to cloud agents like Devin.
  </Step>
  <Step title="Steer">
    Use the **Steering** view to create project rules, user rules, and a Constitution document that shapes how Copilot responds across all your prompts and agents.
  </Step>
  <Step title="Automate">
    Configure **Hooks** to automatically trigger MCP tools, GitHub actions, or custom agents when specs change state, git operations complete, or agent output is ready.
  </Step>
</Steps>

<Note>
  GatomIA requires **GitHub Copilot Chat** to be installed in VS Code. Most features route prompts through Copilot Chat or an ACP-compatible agent like Devin or Gemini CLI.
</Note>
